# Contributing

This repository is published primarily as a read-only educational reference.

The maintainer may disable Issues, pull requests, Discussions, Wiki, Projects,
and GitHub Actions and does not promise support, feature development, review, or
merge of external changes.

You may fork and modify the project under the applicable licenses.

When redistributing a modified copy:

- preserve the project license;
- preserve required third-party notices and license texts;
- clearly describe material changes;
- do not present simulated values, riverbed changes, or display-derived units as
  validated physical measurements or predictions.

Do not report security vulnerabilities publicly.
